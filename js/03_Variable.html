#자바스크립트 변수

  • 자바스크립트 변수에는 타입이 없다 (데이터에는 타입이 있다)
  • 선언 없이도 변수로 만들어서 사용할 수 있다.
  • var : 변수 선언용 키워드
    같은 이름으로 변수를 선언하는 것을 막지 못해 위험하다
  • let : 값을 변경할 수 있는 변수를 선언한다
  • const : 값을 변경할 수 없는 변수를 선언한다(상수)

#변수 규칙

  • 변수 이름에는 알파벳, 숫자, 밑줄(_), $사용
  • 숫자로 시작하면 안됨
  • 동일한 알파벳의 대문자와 소문자는 다른 것 으로 인식한다 (Price, price는 다른변수)
  • 변수 이름을 길게 지을 경우 (_)밑줄 이나 캐멀 케이스가 사용된다
