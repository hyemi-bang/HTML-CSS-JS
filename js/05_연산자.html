# 산술 연산자

  • +,-,*,/,%
  • 자바와 다르게 /가 몫만 구하는 것이 아니라 정확하게 계산을 한다
  • 몫만 구하고 싶을때는 parseInt()로 소수점 아래를 제거해야한다

# 비교연산자

  • <, >, ≤, ≥
  • ==, != : 값이 일치(불일치)하면 true, 타입을 신경쓰지 않음
  • ===, !== : 값과 타입이 일치(불일치)하면 true

# 단항 연산자

  • ++ : 1증가, 위치에 영향을 받음
  • -- : 1감소, 위치에 영향을 받음
  • ! : true <-> false
  • - : 음수는 양수로, 양수는 음수로

# 논리 연산자

  • &&
  • ||

# 할당(대입) 연산자

  • +=
  • -=
  • *=
  • /=
  • %=
  • &=
  • |=

# 삼항 연산자

  • 비교 ? 예 : 아니요;
