# Browser Object Model (BOM)

  • 웹 브라우저를 구조화해놓은 객체
  • window 객체를 통해 웹 브라우저를 직접 제어할 수 있다
  • 자바스크립트에서 window는 생략이 가능하다

# BOM객체

  • window : web browser 최상위 객체
  • window.document : 현재 web page의 HTML 코드 구조 객체 (DOM)
  • window.screen : 모니터 등 기기 화면에 대한 정보
  • window.history : web browser의 page 이동 내역
  • window.location : 현재 web browser가 위치한 경로에 대한 정보
  • window.navigator : web browser에 대한 정보
