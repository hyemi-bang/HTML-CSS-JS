# 이벤트 다루기

  • HTML 속성으로 이벤트 다루기 (onclick, onmouseover, ...)
  • 이벤트 앞에 on을 붙이면 html속성이 된다.
  • 자바스크립트로 해당 요소에 추가하기

# 이벤트 중첩 (버블링, 캡처링)

  • 계층적으로 만들어져있는 요소에서 이벤트가 발생하는 경우 이벤트가 전파된다
  • 버블링: 이벤트가 발생한 요소로부터 부모 요소로 전파되는 것 (이벤트 추가시 기본 설정 되어있다)
  • 캡쳐링: 자식 요소에서 발생한 이벤트가 부모 요소부터 시작하여 자식 요소까지 도달하게 하는 것 (addEventListener() 3번째 매개변수를 true 설정하는 경우 활성화)
  • 이벤트가 발생하는 경우 가장 부모 요소부터 이벤트가 발생한 요소까지 캡처링을 먼저 진행한 후 그 다음에 다시 이벤트가 발생한 요소부터 버블링을 진행한다
