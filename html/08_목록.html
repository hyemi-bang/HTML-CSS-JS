# ul : unordered list 순서 없는 리스트

   # ul, ol 모두 타입(type)을 지정할 수 있다.

- type attr : disc(default), circle, square, none
- none로 type을 없애고 그 자리에 이미지를 넣을 수 있다.

  • 붕어빵
  • 국화빵
  • 계란빵
  • 잉어빵

# ol : ordered list 순서 있는 리스트

- type attr : 1(default), a, A, i, I
- start attr : 시작하는 숫자를 설정할 수 있다.

  1. 군밤
  2. 군고구마
  3. 군귤

# dl : description list 설명이 있는 리스트

- dt(제목), dd (제목보다 들여쓰기) 태그가 있다.
- 들여쓰기를 사용하지 않고, dl 이 가지고 있는 의미만을 사용하기도 한다.
