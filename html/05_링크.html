# 링크를 통해 다른 페이지로 이동하기

1. a태그를 이용
2. 자바스크립트 이용

# a (anchor) tag

- a태그는 전체글의 일부에 링크를 걸 수 있다.
- 자신의 영역을 생성하는 태그가 아니다.
- href 속성에 가고싶은 곳의 주소(url)를 설정해 놓는다.
- 절대경로와 상대경로를 모두 사용할 수 있다.

네이버로 가기
구글로 가기
구글로 가기(http를 뺌) -> file://gogle.com 이라는 파일을 찾는 것임
구글로 가기(http://를 뺌) -> 상대경로로 간다.

# URL 경로의 종류

- naver.com : 상대경로(현재위치)
- ./naver.com : 위와 동일
- /naver.com : 도메인으로부터 시작되는 상대경로
- //naver.com : 프로토콜로부터 시작되는 상대경로
- http://naver.com : URL의 절대경로
