#데이터형

  • Number: 정수와 실수를 모두 포함
  • String: 문자열, ""과 ''을 모두 사용가능
  • Boolean: 참/거짓 타입
  • Array: []을 이용해 배열을 사용
  • Object: {}을 사용해 object를 사용
  • Null: 빈 문자
  • Undefined: 변수가 값을 가지고 있지 않을때
  • Bigint: 정수15자리 넘어갈 경우 끝에 n을 붙인다
  • Symbol: 숫자, 문자열, 불과 같은 원시 데이터 형 심볼은 생성될 때마다 고유한 값을 가지므로, 유일한 식별자로 객체의 속성을 추가하는 데 사용한다

이스케이프 문자

  • \': 작은따옴표
  • \": 큰따옴표
  • \\ : 역슬래쉬
  • \n: 줄바꿈 -> innerHTML, document.write : 브라우저 화면에서 줄바꿈을 할때 HTML줄바꿈태그
    써야함
  • \t: 탭
