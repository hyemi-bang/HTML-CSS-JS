#Position property

  • 포지션 속성을 설정한 요소는 포지션이 설정되지 않은 다른 요소들을 무시하면서 자유롭게 요소 배치 가능하다
  • static(default) - 포지션이 설정되지 않은 상태
  • absolute - 절대적으로 위치를 설정
  • relative - 상대적으로 위치를 설정
  • fixed - 화면에 고정할 위치를 설정
  • sticky - 스크롤이 변하면서 고정될 위치를 설정
  • 포지션이 겹치는 요소가 있따면 z-index 속성으로 요소들의 순서를 정할 수 있다 (어떤 요소가 위에 있게될지)

  • z-index default = 0, 음수로 설정하게 되면 기본요소보다 (적혀있는 글 들)보다 뒤로 가게 된다.
  • 같은 z-index를 갖게 되면 html상 먼저 적혀있는 것이 우선순위를 갖게된다.
    (만약 글이 적힌곳에 백그라운드 설정하면 z-index -로 설정한 요소가 가려진다)

#position: absolute

  • 부모 요소 내부에서 절대적인 위치를 설정
  • 만약 부모 요소에 포지션이 설정되어 있지 않다면 (static이라면) 더 위의 조상으로 올라간다
  • 포지션을 설정하고 나면 top, left, bottom, right를 통해 위치 지정 가능
