# Selector

  • 선택자를 사용해 원하는 HTML요소를 선택해 스타일 적용
  • tagname : 해당 HTML태그를 가진 요소를 모두 선택한다
  • #id : 해당 id 속성값을 가진 요소를 선택한다.
    id 속상값은 해당 웹 페이지 내에서 유일한 값이어야 한다.
    강제성은 없는 기본키 역할이지만 중복하여 사용하지 말기
  • .class : 해당 class 속성값을 가진 요소들을 모두 선택한다
    일반적으로 class속성은 여러 요소에 적용된다.
  • * : 모든 요소 선택
  • html : 모든 요소 선택 (모든 요소의 부모 선택)
  • 공백 : 선택자에서 공백은 모든 자손 요소를 선택한다.
  • > (>) : 모든 자식 요소를 선택
  • , : 한번에 여러개의 선택자를 사용해 선택할 때
