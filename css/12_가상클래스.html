# Pseudo class

  • 선택자 뒤에 :를 붙여 해당 요소의 특정 상황만을 선택하는 것
  • :first-child는 첫 번째 자식요소를 선택하는 것이 아니라 해당 요소가 첫 번째 자식이 된 상황을 선택하는 것이다
  • 구글링 해보기~ https://developer.mozilla.org/en-US/docs/Web/CSS/Pseudo-classes

#a 태그에 특화된 pseudo class 들

  • :hover - 링크에 마우스를 올렸을 때
  • :link - 해당 링크에 방문한 적 없을 때
  • :active - 해당 링크를 클릭하고 있는 상황일 때
  • :visited - 해당 링크에 방문한 적 있을 때
