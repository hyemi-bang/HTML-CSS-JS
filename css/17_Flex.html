# disply: flex

  • 내부에 속한 요소들의 크기를 자동으로 조절해주는 display 속성
  • 레이아웃을 구성할 때 아주 유용한 기능이다
  • display: flex로 설정한 요소는 flex-container가 되고 내부에 속한 요소들은 flex-item이 된다
